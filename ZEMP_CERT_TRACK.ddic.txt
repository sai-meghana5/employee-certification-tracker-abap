Creating a database table in ABAP purely with code using ABAP Dictionary DDL (Data Definition Language) in Eclipse/ADT (ABAP Development Tools). This is the modern way in S/4HANA (instead of SE11).

You do this by creating a DDL Source object with the keyword define table.

@EndUserText.label : 'Employee Certification Tracker'
@AbapCatalog.enhancementCategory : #NOT_EXTENSIBLE
@AbapCatalog.tableCategory : #TRANSPARENT
@AbapCatalog.deliveryClass : #A
@AbapCatalog.dataMaintenance : #ALLOWED
define table zemp_cert_track {

  key emp_id        : abap.char(10);
  key cert_id       : abap.char(10);
  
  cert_name         : abap.char(40);
  issue_date        : abap.dats;
  expiry_date       : abap.dats;
  status            : abap.char(1);       // 'A' = Active, 'E' = Expired
  email             : abap.string(241);   // Employee Email (AD_SMTPADR)

}



USE SE11 
follow the below steps



Table: ZEMP_CERT_TRACK (Custom table for tracking employee certifications)



| **Field Name** | **Data Element / Type** | **Description**                                    |
| -------------- | ----------------------- | -------------------------------------------------- |
| EMP\_ID        | CHAR(10)                | Employee ID                                        |
| CERT\_ID       | CHAR(10)                | Certification ID                                   |
| CERT\_NAME     | CHAR(40)                | Certification Name                                 |
| ISSUE\_DATE    | DATS                    | Certification Issue Date                           |
| EXPIRY\_DATE   | DATS                    | Certification Expiry Date                          |
| STATUS         | CHAR(1)                 | Certification Status ('A' = Active, 'E' = Expired) |
| EMAIL          | AD\_SMTPADR (241)       | Employee Email Address                             |


Key Fields:

EMP_ID
CERT_ID

Technical Settings:
Data Class: APPL0 (Master Data)

Size Category: 0 (Few hundred records)

Delivery Class: A (Application Table)

Table Maintenance: Enabled (via SE11 → Utilities → Table Maintenance Generator)
